Library

The GMH design library is the centralized system for all shared digital assets— components, tokens, styles, and templates— used by staff, students, and affiliates.

It ensures consistency, accelerates collaboration, and provides a reliable foundation for building cohesive digital experiences across the GMH ecosystem.

Built with React. Served on GitHub. This library is available for use and contributions.

Foundations

Design token collections are the styled building blocks behind every component in an organization’s design library.

Geometry

Column grids, pixel grids, and spacing

Typography

Type faces, font weights, and usage

Color

Greyscale, color palettes, and themes

Elevation

Shadows, layering, and depth guidance

Radius

Border radii guidance

Components

In line with atomic design principles, these base components are the next level up from granular design tokens.

Avatars

User profiles brought to life thru headhots and logos.

Buttons

Sizes, actions, types, and interactions

Forms

Base inputs, fields, switches, and theming

Icons

Iconography: nothing more, nothing less.

System

Components to help you document your designs

Templates

Pre-built layout templates and content structures that combine components and styles for common use cases.

Landing Pages

Webpage layouts for campaigns and conversions

Email Templates

Designs for newsletters and communications

Flyers

Printable promotions for events and announcements

Blog Posts

Stylized articles powered by SEO content structure

Social Media

LinkedIn and Bluesky posts optimized for engagement
